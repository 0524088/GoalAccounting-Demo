# 餘額式記帳 APP
針對手機製作的記帳 APP，和一般記帳軟體不同的是，是用"目標"為主體來進行記帳，以便於做金流支出規劃

# 使用技術
主要使用 Javascript 搭配 Bootstrap 5，並使用 Capacitor 和行動裝置進行互動
